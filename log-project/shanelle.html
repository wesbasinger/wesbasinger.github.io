What I’ve learned

by Shanelle Jones

Posted on 04/07/16

First way!

When given a base number and an answer but no exponent use log.

The exponent will be represented by a variable (most likely x).

Find the exponent by bringing x to the front and multiply by the base log. Then divided the answer log by the base log to get x.

Second way!

If given the exponent and answer but no base number take the exponent and add a one on top to make a fraction. Do the same to the answer, insert in the calculator and that will give you your base.

”If at first you don’t succeed, try, try again!”
